title = """We've released SharpEyes, an application for eyetracking and motion-energy features..."""
date = 2026-07-13
+++

We've released <a href="https://gallantlab.org/SharpEyes/">SharpEyes</a>, a desktop application for eyetracking and for building motion-energy features out of stimulus videos.

It finds the pupil in every frame of an eyetracking video, lets you hand-correct the gaze overlaid on the stimulus, recenters the stimulus on gaze to produce a retinotopic video, and computes motion-energy features — all without touching Python. Precompiled Windows and Linux builds are on the <a href="https://github.com/gallantlab/SharpEyes/releases">releases page</a>.

Thanks to <a href="/people/#tianjiao-zhang">Dr. Tianjiao Zhang</a> for developing this tool!
